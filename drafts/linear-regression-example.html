Linear Regression: a simple application

April 01, 2019

In this post we'll show a practical application of the Linear Regression model using an example dataset of houses energy efficiency.
For the task we'll use Python with pandas and statsmodels libraries.

import pandas as pd
import statsmodels.formula.api as smf

The dataset is hosted by UC Irvine Machine Learning Repository from the University of California, Irvine (http://archive.ics.uci.edu/ml/index.php).
The data has 8 features related to building parameters and 2 target variables (heating and cooling requirements). The goal is to predict the energy requirements as a function of the building characteristics.

First, we load the data in a pandas DataFrame; pandas read_excel() method is very handy to load Excel files and allows to read them directly from an URL:

df = pd.read_excel('https://archive.ics.uci.edu/ml/machine-learning-databases/00242/ENB2012_data.xlsx')

Recall the formula for a multiple linear regression model:

$$y = X \beta + \epsilon$$

\(X\) is the feature matrix with n=768 observations and 9 columns (8 features + 1 intercept).
\(y\) is one of the two target variables, suppose it to be heating_load.
\(\beta\) is the unknown 9-vector of parameters.

We know from theory that, under the assumption of linear indipendence of the feature matrix columns, we can compute an OLS estimate for \(\beta\) doing some matrix algebra:

$$\tilde \beta = (X^T X)^{-1} X^T y$$

There is no need to actually implementing the computation because statsmodels already provides a method for that.

statsmodels allows a R-like formula for the definition of a linear regression model, such as

$$target \sim feature_1 + feature_2 + .. + feature_n$$

We generate programmaticaly the regressors part of the formula using a list and the join method:

feature_name_list = [
    'relative_compactness',
    'surface_area',
    'wall_area',
    'roof_area',
    'overall_height',
    'orientation',
    'glazing_area',
    'glazing_area_distribution',
]
regressor_formula = ' + '.join(feature_name_list)
regressor_formula

'relative_compactness + surface_area + wall_area + roof_area + overall_height + orientation + glazing_area + glazing_area_distribution'

We now create an instance of the linear regression model and use the fit method to estimate the parameters from data:

model = smf.ols(formula=f'heating_load ~ {regressor_formula}', data=df)
result = model.fit()
result.summary()

If the distributional hypothesis for the linear regression hold (see the Linear Regression post), the t-values are distributed as \(t_{759}\) (t-Student distribution with 759 degrees of freedom). Being N so large, we can approximate it with a Normal distribution.

What the t-test does is computing the probability \(P(x>|t|)\); if the probability is lower than a chosen threshold (usually 0.05), we reject the null hypothesis \(H_0\) which supposes that \(\beta_j = 0\), concluding that the \(j^{th}\) feature is statistically significant for the model.

The p-value for orientation is very high, so it should probably be removed from the model definition and the model be estimated again.
There are many statistical algorithms that can be used to reduce the number of features (namely subset selection). This will be the topic for another post.
