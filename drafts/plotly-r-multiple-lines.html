Create a multiple line chart with R and Plotly

October 03, 2018

This post shows how to create multiple random time series data and use the Plotly package to draw a line chart.
Plotly (https://plot.ly/r/) is a R package to create modern web-based visualizations via the open source JavaScript graphing library plotly.js. It offers API libraries also for Python and other languages.

First of all, load Plotly into R using:

if(!(require(plotly))){
  install.packages("plotly")
}
require(plotly)

Let's now generate some random data:

n = 50
time <- c(1:n)
# the R function c() combines elements together, and can be used to create vectors.
# In this case it combines the sequence of numbers from 1 to n into a vector.
unifObs <- runif(n)
normObs <- rnorm(n)

The runif(n, min, max) function in R draws n random numbers from a Uniform distribution, i.e. \(X \sim U[min, max]\). By default, \(X \sim U[0, 1]\).

The rnorm(n, mean, sd) function draws n realizations from a Normal distribution, with density:

$$ f(x) = \dfrac{1}{\sqrt{2\sigma^2 \pi}} e^{-\dfrac{(x-\mu)^2}{2\sigma^2}} $$

By default, mean = 0 and sd = 1, i.e. \(X \sim N(0, 1)\).

The plot_ly() function requires the data to be in a dataframe object, which is a common R data structure. We create the dataframe passing 3 vectors of the same length to the data.frame() function:

data = data.frame(time, unifObs, normObs)

Now we can call the actual plot_ly() function to create the chart:

plot_ly(data, x = ~time, y = ~unifObs, name = "uniform",
        type = "scatter", mode = "lines") %>%
  add_trace(y = ~normObs, name = "normal", type = "scatter") %>%
  layout(yaxis = list(title = "observations"))

Key is the use of %>% operator to add new traces to the chart istance.
