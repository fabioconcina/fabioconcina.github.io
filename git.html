Git commands short guide

 

  • Check working directory status:

git status

 

  • View changes to files not yet staged:

git diff

 

  • Add all changes to staging (INDEX):

git add *

 

  • Commit staged files to local repository:

git commit -m "add missing timezone support"

 

  • Push local commits to remote repository:

git push origin master

 

  • Pull commits from remote repository:

git pull

 

  • Delete all your local changes and reset to last commit:

git reset --hard

 

  • Create a new branch:

git branch awesome_branch

 

  • Checkout branch:

git checkout awesome_branch

 

  • Merge a branch into master:

git checkout master

git merge awesome_branch

 

  • Delete branch:

git branch -d awesome_branch
