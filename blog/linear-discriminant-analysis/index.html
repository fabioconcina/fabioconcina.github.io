Linear Discriminant Analysis

May 21, 2019

Linear Discriminant Analysis (LDA) is a classification method which takes a different approach from the one used in Logistic Regression.
Instead of directly modeling the posterior probabilities \(P(Y=k | X=x)\), it uses Bayes theorem to compute them starting from the prior probabilities and conditional density functions of the classes.

More in detail, suppose to have:

  • \(\pi_k\): the prior probability that the response variable \(Y\) belongs to k-th class;
  • \(f_k(x) = P(X=x | K=k)\): the density function of \(X\) conditioned to the information that \(Y\) comes from the k-th class.

We can then apply Bayes theorem to obtain

$$P(Y=k | X=x) = \frac{\pi_k f_k(x)}{\sum_{j=1}^{K}\pi_j f_j(x)} \qquad (1)$$

In general, we don't know \(\pi_k\) and \(f_k(x)\).
\(\pi_k\) can be easily estimated from the training data:

$$\hat \pi_k = \frac{N_k}{N}$$

with \(N_k\) the number of observation from the k-th class and \(N\) the total number of observations.
\(f_k(x)\) is more complex to estimate and we should make some assumptions.
Depending on the hypothesis we make on \(f_k(x)\), we will obtain a different classification method.

If we assume that \(f_k(x)\) follows a Multivariate Gaussian distribution, we obtain the Quadratic Discriminant Analysis (QDA) method:

$$f_k(x) = \frac{1}{(2 \pi)^{p/2} | \Sigma_k |^{1/2}} e^{-\frac{1}{2}(x-\mu_k)^T\Sigma_k^{-1}(x-\mu_k)}$$

If we then make the additional assumption that all the classes share the same covariance matrix, i.e. \(\Sigma_k = \Sigma \enspace \forall k\), we obtain the Linear Discriminant Analysis method.

Substituting in \((1)\) we have

$$P(Y=k | X=x) \propto \pi_k f_k(x) \propto \pi_k e^{-\frac{1}{2}(x-\mu_k)^T\Sigma^{-1}(x-\mu_k)}$$

where we have dropped the factors that not depend on the class \(k\).
Taking the \(\log\) on both sides, we have:

$$\log P(Y=k | X=x) \propto \log\pi_k -\frac{1}{2}(x-\mu_k)^T\Sigma^{-1}(x-\mu_k) \propto \log\pi_k -\frac{1}{2}\mu_k^T\Sigma^{-1}\mu_k + x^T\Sigma^{-1}\mu_k$$

To classify a response, we take the class \(k\) which maximixes the following function:

$$\delta_k(x) = \log\pi_k -\frac{1}{2}\mu_k^T\Sigma^{-1}\mu_k + x^T\Sigma^{-1}\mu_k \qquad (2)$$

The decision boundary is the set of points for which

$$\delta_i(x) = \delta_j(x) \quad i \neq j$$

i.e. the estimated separating boundary between the classes.

We see from \((2)\) that LDA creates linear decision boundaries, hence the name.

Differently, if we remove the hypothesis of common covariance matrix \(\Sigma\) among classes, we obtain the following equation:

$$\delta_k(x) = \log\pi_k -\frac{1}{2}\mu_k^T\Sigma_k^{-1}\mu_k + x^T\Sigma_k^{-1}\mu_k -\frac{1}{2}x^T\Sigma_k^{-1}x -\frac{1}{2} \log |\Sigma_k| \qquad (3)$$

\((3)\) is a quadratic function, so with QDA we will have quadratic decision boundaries.
