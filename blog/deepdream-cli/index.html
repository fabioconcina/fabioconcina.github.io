DEEP LEARNING

DeepDream CLI

January 04, 2020

I recently started playing around with DeepDream, a computer vision algorithm created by Google engineer Alexander Mordvintsev, which utilizes a deep neural network to produced dream-like psychedelic images.

Starting from a Keras example from François Chollet, I built a command line utility (https://github.com/fabioconcina/deepdream-cli) to allow for easier experimentation and hyperparameter tuning.
Following the original script, the utility uses the Inception v3 (https://arxiv.org/pdf/1512.00567.pdf) convolutional neural network model pre-trained on ImageNet dataset, and allows to tune the weight for some specific intermediate layers (which, using Inception v3 original names, are the mixed layers mixed2, mixed3, mixed4, mixed5).

The repository includes an additional script experiment.py, which reads a sequence of experiments from a configurable JSON file and runs in batch the algorithm using the different parameters.
Using this script and a Google Colaboratory notebook with a GPU accelerated runtime, I run some experiments starting from isolating the contribution of each single layer to the final image:

Lower layers seem to generate repetitive geometric patterns, instead higher layers generate more abstract visualizations.
