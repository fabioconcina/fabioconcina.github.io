Logistic Regression

April 07, 2019

Logistic Regression is a popular model for classification. In a classification setting, the response variable is qualitative (categorical), instead of being quantitative as in the case of regression problems.
We would like to predict if a particular observation belongs to a specific class (category).
The simplest case is the one of a binary response variable, i.e. \(Y \in \{0,1\}\). This could represent for example

$$Y=\begin{cases} 1 & \textit{if success,} \\ 0 & \textit{if failure.} \end{cases}$$

Logistic regression models the probability that \(Y\) belongs to a specific class, instead of modeling \(Y\) directly.
Suppose \(X\) being the independent variable, which could be quantitative, taking values in \({\rm I\!R}\), or binary, taking values in \(\{0,1\}\). The output of the model will then be the probability \(P(Y=1|X)\) (and hence \(P(Y=0|X) = 1 - P(Y=1|X)\)).
To model the relationship between the probability \(p(X)\) and \(X\), we can't use a linear relationship, as the predicted probability could fall outside the interval \([0,1]\). We need a function that could map the domain of X into the allowed interval for a probability.
One such function is the logistic function:

$$f(x) = \frac{e^{x}}{1 + e^{x}}, \qquad x \in {\rm I\!R}$$

It's easy to see that

$$\frac{d}{dx} f(x) = \frac{e^x}{(1 + e^x)^2} > 0 \qquad \forall x$$
$$\lim_{x \to -\infty} f(x) = 0, \qquad \lim_{x \to +\infty} f(x) = 1$$

so \(f(x)\) is monotonically increasing from 0 to 1.

We can then model \(p(X)\) as:

$$p(X) = f(\beta_0 + \beta_1 X) = \frac{e^{\beta_0 + \beta_1 X}}{1 + e^{\beta_0 + \beta_1 X}} \tag{1}$$

The parameters \(\beta_0\) and \(\beta_1\) are commonly estimated via maximum likelihood, a general estimation method that tries to find the parameters for which the probability of the observed data is greatest.
The steps for estimation are:

  1. define the likelihood function from the joint probability function of \(Y\);
  2. compute the partial first and second derivatives of the function;
  3. verify that the function has a global minimum using the partial second derivatives;
  4. use a numerical method (Netwon-Raphson) to find the zero of the nonlinear system of equations given by the partial first derivatives.

For a complete derivation, see for example https://czep.net/stat/mlelr.pdf.
Once estimated the parameters \(\hat \beta_0\) and \(\hat \beta_1\), we can start making predictions using

$$P(Y=1 | X=x) = \frac{e^{\hat\beta_0 + \hat\beta_1 x}}{1 + e^{\hat\beta_0 + \hat\beta_1 x}}$$

We could then build a classifier by setting a threshold \(T \in [0,1]\), such that

$$\begin{cases} Y=1 && \textit{if} & P(Y=1 | X=x) \geq T \\ Y=0 && \textit{if} & P(Y=1 | X=x) < T \end{cases}$$

The definition \((1)\) extends naturally in the case of multiple independent variables:

$$p(X_1, X_2, ..., X_p) = f(\beta_0 + \beta_1 X_1 + ... + \beta_p X_p) = \frac{e^{\beta_0 + \beta_1 X_1 + ... + \beta_p X_p}}{1 + e^{\beta_0 + \beta_1 X_1 + ... + \beta_p X_p}}$$

We now highlight a connection with the concept of odds, which are frequently used in gambling and are defined as

$$o = \frac{p}{1-p}$$

The odds take value in \([0, \infty]\), if we define \(o(1)=\infty\). For example, if the probability \(p\) is equal to 0.8, the odds will be \(o(0.8) = \frac{0.8}{0.2} = 4\), and is commonly expressed as 4:1 (4 to 1), meaning that 4 outcomes out of 5 will be positive and 1 outcome out of 5 will be negative.
Rearranging the terms of \((1)\) we have

$$p(X) = \frac{e^{\beta_0 + \beta_1 X}}{1 + e^{\beta_0 + \beta_1 X}} +1 -1 = - \frac{1}{1 + e^{\beta_0 + \beta_1 X}} + 1$$
$$\implies 1-p(X) = \frac{1}{1 + e^{\beta_0 + \beta_1 X}}$$

Hence

$$\frac{p(X)}{1-p(X)} = e^{\beta_0 + \beta_1 X}$$

Taking the logarithm of both sides we finally have

$$\log\left(\frac{p(X)}{1-p(X)}\right) = \beta_0 + \beta_1 X \tag{2}$$

\((2)\) shows that the log-odds (or logit) are linear in the parameters \(\beta_0\) and \(\beta_1\). Increasing the value of X has the effect of linearly increasing the log-odds of success.
