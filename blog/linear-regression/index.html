Linear Regression

March 24, 2019

Linear Regression is one of the simplest yet explainable models to describe the relationship between quantitative variables. More advanced statistical learning models can be seen as a generalization of this approach.
In this model, one or more indipendent variables (explanatory variables, regressors) are used to predict one dependent variable (response variable, regressand).
The model assumes that there is approximately a linear relationship between indipendent and dependent variables.

The scenario with a single indipendent variable is called simple linear regression:

$$Y = \beta_0 + \beta_1 X + \epsilon$$

\(Y\) is the response variable and \(X\) is the explanatory variable.
\(\epsilon\) is an error term, a random variable used to model the deviation from the regression line observed in the data. \(\beta_0\) and \(\beta_1\) are called parameters. They are unknown and we must estimate them from the data. This step represents the training process for the model. To estimate the parameters, the most common algorithm is the ordinary least squares (OLS). The method tries to minimize the sum of squares of the differences between the actual observations of the response variable and the predicted response using the model. The estimated regression line is the best fit to the data in the sense of least squares; this can be different from the actual population line, since we only have available a sample of the population to estimate the model.
Once the parameters are estimated, we can use the model to predict the response variable given a value of the explanatory variable. We can also explain how the explanatory variable influences the response variable (does it change when the indipendent variables changes? how much does it change?). Notice that the linearity is in the parameters, but not in the indipendent variables (\(Y = \beta_0 + \beta_1 X^2 + \epsilon\) is still a linear model).

The case with multiple indipendent variables is called multiple linear regression:

$$Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + ... + \beta_p X_p + \epsilon$$

Supposing to have N observations, the last formula can conveniently represented with matrix linear algebra:

$$y = X \beta + \epsilon$$

\(X\) is a N x (p+1) matrix where we have p indipendent variables plus 1 constant term.
\(y\) is a N-vector of response variables, \(\beta\) is a (p+1)-vector of parameters and \(\epsilon\) is a N-vector of errors.
We now minimize the sum of squares using this linear algebra representation.
We can then write the error term as a function of \(\beta\) as

$$\epsilon(\beta) = y - X \beta$$

The residual sum of squares is

$$RSS(\beta) = (y - X \beta)^T(y - X \beta)$$

Using some calculus we can compute the derivative of the quadratic function:

$$\frac{\partial RSS}{\partial \beta} = -2X^T (y - X \beta)$$
$$\frac{\partial ^ 2 RSS}{\partial^2 \beta} = 2 X^T X$$

If \(X^T X\) has full rank, we can compute the inverse and \(X^T X\) is positive definite, so the function has a global minimum where the first derivative is equal to zero:

$$X^T (y - X \beta) = 0$$

The unique solution is

$$\tilde \beta = (X^T X)^{-1} X^T y$$

\(\tilde \beta\) is an estimator for \(\beta\).
The predicted value for the response variable is then

$$\tilde y = X \tilde \beta = X (X^T X)^{-1} X^T y = H y$$

H is usually called hat matrix.
If \(X\) has not full rank, i.e. some columns are not linearly indipendent, as in the case of perfect correlation, some strategy is necessary to remove the columns.

We should now make some probabilistic hypothesis on the random variables used in the model, so that we can derive additional results.

  1. Errors variance is constant (homoschedasticity), i.e.
    $$Var(\epsilon_i) = \sigma^2 \quad \forall i$$
  2. Errors have expected value equal to 0, i.e.
    $$E(\epsilon_i) = 0 \quad \forall i$$
  3. Errors are uncorrelated, i.e.
    $$Cov(\epsilon_i, \epsilon_j) = 0 \quad \forall i,j, \: i \neq j$$
  4. The conditional expectation of the response variable is linear in the indipendent variables, i.e.
    $$E(Y | X) = X \beta$$

We can require a more stronger condition of \((1)+(2)+(3)\), which is

$$\epsilon \sim N(0, \sigma^2) \quad i.i.d. \qquad (*)$$

The additional assumptions made by \((*)\) are that errors follow a Gaussian distribution and are indipendent (a stronger assumption than uncorrelated).

Under \((1)+(2)+(3)+(4)\) we have

$$E(\tilde \beta | X) = (X^T X)^{-1} X^T E(y | X) = (X^T X)^{-1} X^T X \beta = \beta$$

and

$$Var(\tilde \beta | X) = (X^T X)^{-1} X^T Var(\beta | X) X ((X^T X)^{-1})^T =$$
$$= (X^T X)^{-1} X^T X ((X^T X)^T)^{-1} \sigma^2 = (X^T X)^{-1} \sigma^2$$

So \(\tilde \beta\) is an unbiased estimator for \(\beta\). Under \((*)\) we have then

$$\tilde \beta \sim N(\beta, (X^T X)^{-1} \sigma^2)$$

To estimate \(\sigma^2\) we can use the following estimator:

$$\tilde \sigma^2 = \frac{1}{N-p-1} \sum_{i=1}^{N} (y_i - \tilde y_i)^2$$

The Gauss-Markov theorem states that, under the above hypothesis, the OLS estimator \(\tilde \beta\) has the minimum variance among unbiased estimators for \(\beta\).
It is still possible though to find a biased estimator with lower variance.

We can then use the results to compute confidence intervals and perform hypothesis tests on the parameters.
To compute a confidence interval for the parameters we can use the distributional information on \(\tilde \beta\):

$$\left(\tilde \beta_j - t^{(1-\alpha)}v_j^{1/2} \tilde \sigma, \quad\tilde \beta_j + t^{(1-\alpha)}v_j^{1/2} \tilde \sigma \right)$$

where \(t_j^{(1-\alpha)}\) is the \(1-\alpha\) percentile of the Student's t-distribution with N-p-1 degrees of freedom and \(v_j\) is the j-th diagonal element of \(X\) matrix.
If we know the real value of \(\sigma\), or the sample size N is sufficiently large, we can use the percentile of a Normal distribution \(z_j^{(1-\alpha)}\) instead.
We can construct an hypothesis test as follows:

$$H_0: \qquad \beta_j = 0$$
$$H_1: \qquad \beta_j \neq 0$$

We use as test statistics the standardized coefficient (or z-score), a random variable defined as

$$z_j = \frac{\tilde \beta_j}{\tilde \sigma \sqrt{v_j}}$$

Under \(H_0\), \(z_j \sim t_{N-p-1}\) (Student's t-distribution with N-p-1 degrees of freedom).
A large value of \(z_j\) leads to reject the null hypothesis \(H_0\), which implies that the parameter \(\beta_j\) is significantly different from zero, hence there is a relationship between the \(X_j\) variable and the response variable.

Considering the easier case of simple linear regression, we can explicitly write the estimators for the standard error of the parameters:

$$SE(\tilde \beta_0)^2 = \sigma^2 \left[\frac{1}{N} + \frac{\bar x^2}{\sum_{i=1}^{N} (x_i - \bar x)^2} \right]$$
$$SE(\tilde \beta_1)^2 = \frac{\sigma^2}{\sum_{i=1}^{N}(x_i - \bar x)^2}$$

Using the estimator \(\tilde \sigma\) we can then derive numeric values for the standard errors.
We can compute confidence intervals using

$$\left(\tilde \beta_j - z^{(1-\alpha)} SE(\tilde \beta_j), \quad \tilde \beta_j + z^{(1-\alpha)} SE(\tilde \beta_j) \right) \qquad j=0,1$$

If we choose a 95% confidence interval, \(\alpha=0.025\) and \(z^{(1-\alpha)}=1.96\).
