Linear Regression: OLS

March 24, 2019

Linear Regression is one of the simplest yet explainable models to describe the relationship between quantitative variables. More advanced statistical learning models can be seen as a generalization of this approach.
In this model, one or more indipendent variables (explanatory variables, regressors, features) are used to predict one dependent variable (response variable, regressand).
The model assumes that there is approximately a linear relationship between indipendent and dependent variables.

The scenario with a single indipendent variable is called simple linear regression:

$$Y = \beta_0 + \beta_1 X + \epsilon$$

\(Y\) is the response variable and \(X\) is the explanatory variable.
\(\epsilon\) is an error term, a random variable used to model the deviation from the regression line observed in the data.
\(\beta_0\) and \(\beta_1\) are called parameters. They are unknown and we must estimate them from the data. This step represents the training process for the model.

To estimate the parameters, the most common algorithm is the ordinary least squares (OLS). This method tries to minimize the sum of squares of the differences between the actual observations of the response variable and the predicted response using the model.
The estimated regression line is the best fit to the data in the sense of least squares; this can be different from the actual population line, since we only have available a sample of the population to estimate the model.

Once the parameters are estimated, we can use the model to predict the response variable given a value of the explanatory variable. We can also explain how the explanatory variable influences the response variable (does it change when the indipendent variables changes? how much does it change?).
Notice that the linearity is in the parameters, but not in the indipendent variables (\(Y = \beta_0 + \beta_1 X^2 + \epsilon\) is still a linear model).

The case with multiple indipendent variables is called multiple linear regression:

$$Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + ... + \beta_p X_p + \epsilon$$

Supposing to have N observations, the last formula can conveniently represented with matrix linear algebra:

$$y = X \beta + \epsilon$$

\(X\) is a N x (p+1) matrix where we have p indipendent variables plus 1 constant term.
\(y\) is a N-vector of response variables, \(\beta\) is a (p+1)-vector of parameters and \(\epsilon\) is a N-vector of errors.
We now minimize the sum of squares using this linear algebra representation.
We can then write the error term as a function of \(\beta\) as

$$\epsilon(\beta) = y - X \beta$$

The residual sum of squares is

$$RSS(\beta) = (y - X \beta)^T(y - X \beta)$$

Using some calculus we can compute the derivative of the quadratic function:

$$\frac{\partial RSS}{\partial \beta} = -2X^T (y - X \beta)$$
$$\frac{\partial ^ 2 RSS}{\partial^2 \beta} = 2 X^T X$$

If \(X^T X\) has full rank*, we can compute the inverse and \(X^T X\) is positive definite, so the function has a global minimum where the first derivative is equal to zero:

$$X^T (y - X \beta) = 0$$

The unique solution is

$$\tilde \beta = (X^T X)^{-1} X^T y \tag{1}$$

\(\tilde \beta\) is an estimator for \(\beta\).

\((1)\) is a closed form solution for the model parameters.
In general, in a statistical learning model we need to estimate the parameters through an iterative process.

Note that the OLS derivation for the parameters makes no probabilistic assumptions on the random variables used in the model.

(*) If \(X\) has not full rank, i.e. some columns are not linearly indipendent, as in the case of perfect correlation, some strategy is necessary to remove the columns.
