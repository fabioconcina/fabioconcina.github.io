Linear Regression: probabilistic assumptions

March 24, 2019

To derive additional results on the parameters estimator \(\tilde \beta\) of the Linear Regression model, suppose the following hypotheses to hold:

  1. Errors variance is constant (homoschedasticity), i.e.
    $$Var(\epsilon_i) = \sigma^2 \quad \forall i$$
  2. Errors have expected value equal to 0, i.e.
    $$E(\epsilon_i) = 0 \quad \forall i$$
  3. Errors are uncorrelated, i.e.
    $$Cov(\epsilon_i, \epsilon_j) = 0 \quad \forall i,j, \: i \neq j$$
  4. The conditional expectation of the response variable is linear in the independent variables, i.e.
    $$E(Y | X) = X \beta$$

We can require a more stronger condition of \((1)+(2)+(3)\), which is

$$\epsilon \sim N(0, \sigma^2) \quad i.i.d. \qquad \tag{*}$$

The additional assumptions made by \((*)\) are that errors follow a Gaussian distribution and are independent (a stronger assumption than uncorrelated).

Under \((1)+(2)+(3)+(4)\) we have

$$E(\tilde \beta | X) = (X^T X)^{-1} X^T E(y | X) = (X^T X)^{-1} X^T X \beta = \beta$$

and

$$Var(\tilde \beta | X) = (X^T X)^{-1} X^T Var(\beta | X) X ((X^T X)^{-1})^T =$$
$$= (X^T X)^{-1} X^T X ((X^T X)^T)^{-1} \sigma^2 = (X^T X)^{-1} \sigma^2$$

So \(\tilde \beta\) is an unbiased estimator for \(\beta\) and has conditional variance \((X^T X)^{-1} \sigma^2\).

Under \((*)\) we have then

$$\tilde \beta \sim N(\beta, (X^T X)^{-1} \sigma^2)$$

The Gauss-Markov theorem states that, under the above hypothesis, the OLS estimator \(\tilde \beta\) has the minimum variance among unbiased estimators for \(\beta\).
It is still possible though to find a biased estimator with lower variance.

To estimate \(\sigma^2\) we can use the following estimator:

$$\tilde \sigma^2 = \frac{1}{N-p-1} \sum_{i=1}^{N} (y_i - \tilde y_i)^2$$


Confidence Intervals

To compute a confidence interval for the parameters we can use the distributional information on \(\tilde \beta\):

$$\left(\tilde \beta_j - t^{(1-\alpha)}v_j^{1/2} \tilde \sigma, \quad\tilde \beta_j + t^{(1-\alpha)}v_j^{1/2} \tilde \sigma \right)$$

where \(t_j^{(1-\alpha)}\) is the \(1-\alpha\) percentile of the Student's t-distribution with N-p-1 degrees of freedom and \(v_j\) is the j-th diagonal element of \(X\) matrix.
If we know the real value of \(\sigma\), or the sample size N is sufficiently large, we can use the percentile of a Normal distribution \(z_j^{(1-\alpha)}\) instead.

Considering the easier case of simple linear regression, we can explicitly write the estimators for the standard error of the parameters:

$$SE(\tilde \beta_0)^2 = \sigma^2 \left[\frac{1}{N} + \frac{\bar x^2}{\sum_{i=1}^{N} (x_i - \bar x)^2} \right]$$
$$SE(\tilde \beta_1)^2 = \frac{\sigma^2}{\sum_{i=1}^{N}(x_i - \bar x)^2}$$

Using the estimator \(\tilde \sigma\) we can then derive numeric values for the standard errors.
We can compute confidence intervals using

$$\left(\tilde \beta_j - z^{(1-\alpha)} SE(\tilde \beta_j), \quad \tilde \beta_j + z^{(1-\alpha)} SE(\tilde \beta_j) \right) \qquad j=0,1$$

If we choose a 95% confidence interval, \(\alpha=0.025\) and \(z^{(1-\alpha)}=1.96\).


Hypothesis test

We can construct an hypothesis test on a single parameter as follows:

$$H_0: \qquad \beta_j = 0$$
$$H_1: \qquad \beta_j \neq 0$$

We use as test statistics the standardized coefficient (or z-score), a random variable defined as

$$z_j = \frac{\tilde \beta_j}{\tilde \sigma \sqrt{v_j}}$$

Under \(H_0\), \(z_j \sim t_{N-p-1}\) (Student's t-distribution with N-p-1 degrees of freedom).
A large value of \(z_j\) leads to reject the null hypothesis \(H_0\), which implies that the parameter \(\beta_j\) is significantly different from zero, hence there is a relationship between the \(X_j\) variable and the response variable.
